Открытие из файла и сохранение в файл


Открытие из файла.


Так же у пользователя имеется возможность открыть задачу из файла. Для этого необходимо выбрать пункт меню "Файл" и из выпадающего окна выбрать пункт "Открыть задачу из файла"







В появившемся меню выбираем файл задачи. Программа преобразует задачу в нужный формат, расставит коэффициенты и параметры.





Документация по содержанию текстовой задачи:

  1. Файл должен иметь формат .txt (Текстовый файл)

  2. Вводятся только коэффиценты

  3. Коэффиценты должны быть разделены пробелом

  4. Если коэффициент отрицателен, перед ним ставится минус.

  5. Чтобы указать минизацию или максимизацию задачи, следует ввести "->min" или "->max" после целевой функции, например "1 4 1 -4 ->min"

  6. Программа так же считывает дробные значения.
    Дробь должна быть разделена дробной чертой в формате 1/3

  7. В файле задачи не должно быть никаких лишних символов.
    После окончания целевой функции не должно быть пробела.
    После окончания любого из ограничений не должно быть пробела.
    Задача оканчивается последним коэффициентом! В конце задачи не должно быть пробела. Или переноса на следующую строку*


*Программа всё же учитывает некоторые опечатки и ошибки ввода и чистит вводные данные от некоторых лишних символов, но всё же желательно руководствоваться правилами указаными выше - тогда открытие задач из файла не создаст никаких проблем.
